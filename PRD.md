## 1. 产品概述

### 1.1 产品定位

「塔罗指引」是一款纯前端、重视抽牌仪式感与结构化解读的在线塔罗工具。用户选择牌阵和问题领域，可选填具体问题，随后完成洗牌、翻牌和阅读。系统依据牌阵位置、牌面正逆位、问题领域及牌间关系规则生成解读。

无需注册，不上传用户问题，可在完成首次加载后离线使用。

### 1.2 目标用户

- 对塔罗感兴趣但缺少系统知识的普通用户
- 希望借助象征性内容整理思路的人
- 手机端优先，同时支持桌面浏览器与 PWA 安装

### 1.3 核心价值

- 无需注册，打开即用
- 保留洗牌、翻牌的仪式感
- 不止复述单牌释义，而是提供位置化、领域化和关系化解读
- 用户问题仅保留在本地

### 1.4 产品边界

> 塔罗解读仅用于娱乐与自我反思，不构成医疗、法律、财务或其他专业建议。

## 2. 核心用户流程

```text
选择牌阵 → 选择领域并提问 → 洗牌与逐张翻牌 → 综合解读
```

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 产品介绍、边界说明、开始按钮 |
| `/select` | 牌阵选择 | 选择 3 种牌阵之一 |
| `/question/:spreadId` | 问题设置 | 选择问题领域，可选填具体问题 |
| `/reading` | 占卜场景 | 洗牌、翻牌、单牌详情和综合结果 |

## 3. 功能需求

### 3.1 三种牌阵

**三张牌 · 时间之箭**（3 张，线性排列）

| 位置 | 含义 |
|------|------|
| 过去 | 已发生的背景或问题来源 |
| 现在 | 当前状况或核心挑战 |
| 未来 | 按当前路径可能出现的发展方向 |

**六芒星 · 星芒指引**（7 张，六芒星布局）

| 位置 | 含义 |
|------|------|
| 核心 | 问题本质或当前核心状态 |
| 目标 | 期待的结果或理想方向 |
| 助力 | 可利用的资源和支持 |
| 阻碍 | 困难、限制或需面对的代价 |
| 根源 | 深层原因或未被看见的影响 |
| 建议 | 可考虑的态度或行动方向 |
| 结果 | 按当前路径可能出现的走向 |

**二选一 · 抉择之路**（5 张，分叉布局）

| 位置 | 含义 |
|------|------|
| 现状 | 当前选择的核心状态 |
| A 的优势 | 路径 A 的积极面 |
| A 的挑战 | 路径 A 的风险或代价 |
| B 的优势 | 路径 B 的积极面 |
| B 的挑战 | 路径 B 的风险或代价 |

### 3.2 问题设置

用户选择一个问题领域（必选）：
- 感情关系 / 事业工作 / 学业成长 / 财务现实 / 人际家庭 / 一般问题

具体问题可选填（最大 200 字）。二选一牌阵需填写选项 A 和 B（各最多 40 字）。

### 3.3 抽牌与洗牌

1. 点击"开始抽牌"后按钮立即禁用，防止重复提交
2. 从 78 张牌中无放回抽取指定数量
3. 每张牌独立 50% 概率逆位
4. 播放约 1.2–1.8 秒洗牌动画
5. 牌背按牌阵位置出现

### 3.4 翻牌与解读

- 可按任意顺序逐张翻牌
- 卡牌使用真实 `<button>`，支持鼠标、触控、键盘
- 点击后播放 3D 翻转动画
- 逆位时插画旋转 180°，文字保持正常可读
- 单牌详情：牌名、正/逆位、核心含义、领域补充说明、行动建议

### 3.5 综合解读

所有牌翻开后展示结构化解读（三张牌 250-450 字、六芒星 500-800 字、二选一 350-600 字），包含：
1. 问题领域回顾
2. 整体主题
3. 各位置领域化解读
4. 关键牌面关系（大阿卡纳集中、主导花色、正逆位格局、位置呼应等）
5. 可执行的反思或行动建议

### 3.6 结果操作

- "重新占卜"：清空状态，返回牌阵选择
- "复制牌面信息"：复制问题、牌阵、位置、牌名和正逆位

### 3.7 会话恢复

使用 sessionStorage 保存占卜状态。刷新后恢复牌阵、问题、抽牌结果和翻牌进度。

## 4. 视觉风格

几何黑金风格：
- 背景曜石黑 `#080908`，主金色 `#c9a050`，文字奶油白 `#f2ead7`
- 78 张统一几何黑金 SVG 插画，2:3 比例
- 移动端六芒星使用缩略牌阵 + 详情面板
- 翻牌动画仅使用 `transform` 和 `opacity`

---

## 5. 隐私与离线

- 用户输入和抽牌结果只保存在当前浏览器
- 无 Cookie、无追踪、无第三方
- PWA 安装后离线可用
